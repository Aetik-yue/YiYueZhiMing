# 以越之名 1.0.5 更新日志

发布日期：2026-05-11

## 更新内容

- 新增统一 DeepSeek API 服务层，支持 API Key 本机加密保存、请求校验、错误处理和请求日志记录。
- 百宝箱新增运势入口，包含每日签运与每日塔罗牌，并支持本地历史记录查询。
- 底部导航新增 AI 聊天入口，支持 DeepSeek 对话、聊天历史保存、发送状态与错误提示。
- 相册入口迁移到百宝箱，底部导航移除相册入口。
- 设置页新增 DeepSeek API Key 管理；关于版本号跟随发布版本。
- 日期卡片支持长按编辑和删除；相册与设置页移除多余说明文字。
